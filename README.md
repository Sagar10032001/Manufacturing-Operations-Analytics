# Manufacturing Operations Analysis

## Project Overview

This project analyzes manufacturing operations data to identify opportunities for improving production efficiency, machine performance, workforce utilization, and operational decision-making.

The project follows a structured data analytics workflow, starting from understanding the business problem and dataset, through data cleaning and analysis, and ending with interactive dashboards and business recommendations.

---

## Business Problem

Manufacturing organizations generate large amounts of operational data every day. However, without proper analysis, it is difficult to identify inefficiencies, monitor production performance, and support data-driven decision-making.

This project aims to analyze manufacturing operations data to answer key business questions related to production performance, machine efficiency, workforce utilization, maintenance, quality risk, and operational trends.

---

## Project Objectives

- Understand the manufacturing dataset.
- Profile and assess data quality.
- Clean and prepare the data for analysis.
- Perform exploratory analysis using SQL and Excel.
- Build an interactive Power BI dashboard.
- Generate business insights and recommendations.
